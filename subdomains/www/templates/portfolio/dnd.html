FoundryVTT

I use foundry virtual table top to power dnd.dumbfucks.club. It is like https://roll20.net except you purchase access to the code. For $40 you can then freely mod the javascript of the website yourself or download plugins from other developers. Only downside (which for people like me is arguably a feature) is you have to self host this website.
